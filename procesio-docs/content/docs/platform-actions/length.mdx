## Overview

The `Length` action returns the number of elements in a given list.

![](/images/docs/eo0JVAKyG7jssqCNY8YxG_image.png)

### Parameters

### Inputs

- **input a list of elements** (`list<object>`)

### Outputs

- **result&#x20;**(`object`): The number (`integer`) of elements in the input list
