Returns the value at a given index from a list.&#x20;

**Obs:** Indexing is **zero-based**. The first element is at index **0**.

## Inputs & Outputs

- **\[ I ] Input a list:** Array/List (`IEnumerable<object>`) from which to extract the element.
  - **Examples:** `[1, 2, 3]`, `["a","b","c"]`, mixed types are allowed.
- **\[ I ] Input the index of the element:** Integer (0-based).
  - **Tooltip:** *Starting from zero*
  - **Constraints:** Must be between `0` and `list.Count - 1`.
- **\[ O ] Result:** The element found at the specified index. Type matches the element stored in the list (e.g., number, string, object).

## Example Usage

- Drag **Get X Element** onto the canvas to get familiarized with it.
- Declare two variables (or map existing ones):
  - `myList` → `[10, 20, 30, 40]`
  - `myIndex` → `2`
- Configure the action:
  - **Input a list** = `myList`
  - **Input the index of the element** = `myIndex`
  - **Result** → `selectedValue`
- **Save & Run** the process and then **Check Instance** to see the output.

**Expected result:** `selectedValue = 30`

## Use Cases

- Picking an item from a generated list (e.g., the N-th record).
- Accessing the “current” element by an index computed upstream.
- Extracting a specific configuration value from an options array.
