The **Beta&#x20;**&#x66;older contains actions that can be used while working on different scenarios:

&#x20; 1\.    **Duplicate identification** ➜ Identify duplicates in a list based on criteria.

&#x20; 2\.    **RegEx Replace** ➜ Replace RegEx matches from a string.

&#x20; 3\.    **RegEx Extract** ➜ Extract a list of RegEx matches from a string.

&#x20; 4\.    **Multiply** ➜ Multiplication  operation for two objects.

&#x20; 5\.    **DeDuplicate list** ➜ Identify duplicates in a simple list, based on criteria.

&#x20; 6\.    **Merge lists** ➜ Merge the data from two lists into one.
